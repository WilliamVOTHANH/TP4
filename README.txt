
1/ Un bot est un logiciel informatique permettant de lancer des commandes pour effectuer certaines tâches spécifiques, il peut être considéré comme un utilisateur à part entière.

2/ Un logging est la gestion de logs, les logs sont tout les enregistrements séquentiels d'actions effectuées dans un fichier. Les logs permettent de retracer ce que l'utilisateur a fait, comme une sorte d'historique d'évènement.


La commande pour lancer le bot est : python client.py -c ! --config config.json
